# VolentScript

**ActionScript, reborn** — a statically-typed, ES4-inspired language that
compiles to native executables via LLVM.

## → The code lives at [ZlatanOmerovic/volentscript](https://github.com/ZlatanOmerovic/volentscript)

Active development happens on the repo above. This org is a reserved home for
the project — watch or star the main repo to follow along.
